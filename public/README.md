# Carpeta Public

Coloca aquí los archivos estáticos que se servirán directamente:

- `profile.jpg` - Tu foto de perfil (recomendado: 500x500px o más, formato JPG/PNG)
- `project-placeholder.jpg` - Imágenes de proyectos (recomendado: 800x600px o más)
- `favicon.ico` - Icono del sitio web

Los archivos en esta carpeta estarán disponibles en la raíz del sitio (ej: `/profile.jpg`).
