# Currículum Web

Currículum web profesional desarrollado con Vue.js y Nuxt 3.

## Características

- Diseño moderno, limpio y responsive
- Tres páginas principales: Inicio, Proyectos y Contacto
- Componentes reutilizables
- Navegación intuitiva


## Estructura

- `/pages` - Páginas principales
- `/components` - Componentes reutilizables
- `/assets` - Estilos y recursos estáticos
- `/layouts` - Layouts de la aplicación
